# Cursor Taste Rule

Use the design taste library before generating UI.

## Instructions

- Check Visual-references.canvas and principles first.
- Use references as abstraction, not as clone targets.
- Avoid generic AI-product design patterns.
- Name the references influencing the output.
- Prefer calm, structured, high-quality product UI.
- If the direction is unclear, ask for a reference before inventing a style.
